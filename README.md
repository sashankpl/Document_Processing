# Tech Stack Involved
- Frontend - HTML, CSS, Javascript
- Backend - Flask(Python)
- Cloud Services - AWS Textract(Extracting text), AWS Cognito(Authentication), AWS Lambda(Serverless Compute)
- APIs - Google Gemini API for Q&A
